
# OrgFinance FULL Version

## Instalasi
1. Copy ke htdocs
2. Buat database 'orgfinance'
3. Import database/schema.sql lalu seed.sql
4. Edit koneksi di app/config/db.php jika perlu
5. Buka: http://localhost/OrgFinance_FULL/public/
